Technical Requirements Document (TRD)
Project Title

DebugGPT: AI-Powered Code Error Explanation & Debugging Assistant

1. Technical Overview

DebugGPT is an AI-powered debugging platform that combines:

Fine-Tuned Mistral 7B Instruct
Retrieval-Augmented Generation (RAG)
ChromaDB Vector Store
FastAPI Backend
Streamlit Frontend
Docker-based Deployment

The system analyzes source code, detects bugs, retrieves similar debugging cases, and generates detailed explanations and fixes.

5. API Specifications
Endpoint 1
POST /analyze
Purpose

Analyze source code.

Request
{
  "language": "cpp",
  "code": "source code here"
}
Response
{
  "issues":[...],
  "complexity":"O(n)",
  "recommendations":[...]
}
Endpoint 2
POST /debug
Purpose

Debug source code.

Request
{
  "language":"cpp",
  "code":"..."
}
Response
{
  "error":"Detected issue",
  "explanation":"...",
  "fix":"..."
}
Endpoint 3
POST /explain
Purpose

Explain pasted error.

Request
{
  "error":"Segmentation Fault"
}
Response
{
  "technical":"...",
  "beginner":"...",
  "analogy":"..."
}
Endpoint 4
POST /optimize
Purpose

Optimize code.

Request
{
  "code":"..."
}
Response
