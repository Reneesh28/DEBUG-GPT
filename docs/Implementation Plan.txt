Resume Description

Built DebugGPT, an AI-powered debugging assistant using Mistral 7B, LoRA fine-tuning, ChromaDB RAG, FastAPI, and Streamlit. The system automatically analyzes code, explains compiler/runtime errors, suggests fixes, performs optimization analysis, and generates multi-level educational explanations for developers.

Recommended Actual Build Order (Important)

Don't start with fine-tuning.

Follow this order:

1. Project Setup
2. Dataset Collection
3. ChromaDB RAG
4. Rule-Based Analyzer
5. FastAPI Backend
6. Streamlit Frontend
7. End-to-End MVP Testing
8. LoRA Fine-Tuning
9. Dockerization
10. Documentation

This way, by Day 4–5 you already have a working DebugGPT. Fine-tuning becomes an enhancement rather than a blocker, which is much safer for a 1-week academic project.
