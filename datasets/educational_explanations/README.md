# Educational Explanations Dataset

This dataset partition focuses on explaining language features, idioms, and paradigms rather than fixing broken code. 

## Supported Languages & Concepts

* **Python:** Object lifecycle (`__init__` vs `__new__`), variable length arguments (`*args`, `**kwargs`), Context Managers (`with`), Generators (`yield`), and Decorators.
* **C++:** Value Categories (`lvalue`/`rvalue`), Dynamic Dispatch & Polymorphism (`virtual`), Smart Pointers (`std::unique_ptr`/`shared_ptr`), Templates & Generic Programming, and Resource Acquisition Is Initialization (RAII).

## Purpose
Used to train the model to answer "How does this work?" or "What is the best way to do X?". It establishes the AI as a tutor capable of providing strong analogies and technical breakdowns of abstract concepts.
